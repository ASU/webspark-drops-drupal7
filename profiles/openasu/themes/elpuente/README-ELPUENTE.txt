This theme includes bootstrap 3.1.0.2, to match up with OpenASU's version.

Bootstrap-SASS came from https://github.com/twbs/bootstrap-sass/tree/3.1-stable

Installed via Compass without Gemfile: compass create my-new-project -r bootstrap-sass --using bootstrap

Compass watch will compile the SASS

Read the README.md file at https://github.com/twbs/bootstrap-sass/tree/v3.1.0%2B2 for more
