DESCRIPTION
--------------------------
Module to add the AJAX Solr Library (https://github.com/evolvingweb/ajax-solr).

INSTALLATION
--------------------------
Install as usual, see http://drupal.org/node/70151 for further information.

Download the most recent AJAX Solr library from
https://github.com/evolvingweb/ajax-solr/tarball/master.

Once downloaded, decompress the archive and rename the library's root directory
to ajax_solr and place it in sites/all/libraries (or wherever you store your
Drupal Libraries).

At the time of this module's creation, there was no official release for
AJAX Solr. We worked with a version downloaded on 2015/7/29. Visit
https://github.com/evolvingweb/ajax-solr for access to various versions of
the library.
