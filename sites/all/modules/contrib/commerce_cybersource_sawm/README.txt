INTRODUCTION
------------

Commerce CyberSource SAWM (Secure Acceptance Web/Mobile) integration
for the Drupal Commerce payment and checkout system.

INSTALLATION
------------

 * Install the CyberSource SAWM module.
 * Then enable the Payment Rule for CyberSource Web/Mobile via Store > Configuration >
   Payment settings and edit the enabling action to use your CyberSource API
   credentials.
